# 个人健康管理系统 java763

## 项目概述

个人健康管理系统是一款基于前后端分离架构设计的系统，旨在帮助用户全面管理个人健康信息。系统通过数据分析，为用户提供健康评估和运动知识推荐，助力用户更好地掌握和提升自身健康状况。

## 技术栈

- 后端：Spring Boot + MyBatis-Plus
- 前端：Vue2 + ElementUI + ECharts + Vuex
- 数据库：MySQL、Redis

## 功能模块

### 用户管理

- 注册
- 登录
- 系统可视化统计信息首页

### 系统管理

- 用户管理：增删改查用户
- 角色管理：普通用户与超级管理员

### 健康信息管理

- 运动知识管理：新增、修改、删除和搜索运动知识
- 运动详情管理：新增、修改、删除和搜索运动详情
- 身体信息管理：修改、删除和搜索用户身体信息
- 身体信息上传：上传用户身体各项信息

### 健康评估与知识

- 健康评估：通过ECharts进行健康信息可视化统计
- 运动知识：搜索和查看各项运动相关内容

### 身体健康信息管理

- 编辑和删除健康信息

## 系统角色

- 普通用户：可查看、管理个人信息，进行健康评估等
- 超级管理员：拥有用户管理和系统管理的权限

## 运行环境

- 后端：Java 1.8+，MySQL 5.7+，Redis 3.2+
- 前端：现代浏览器

## 目录结构

```
project/
├── backend/          # 后端代码
├── frontend/         # 前端代码
└── database/         # 数据库脚本
```

## 部署步骤

1. 准备运行环境，包括Java、MySQL和Redis的安装与配置。
2. 部署后端服务，执行数据库脚本，初始化数据。
3. 部署前端服务，与后端服务对接。

## 核心亮点

- 基于Spring Boot和Vue2的前后端分离架构，易于维护和扩展。
- 使用MyBatis-Plus提升数据库操作的效率。
- 结合ECharts进行健康数据可视化展示，直观呈现用户健康状态。
- 提供全面的用户身体信息管理，支持用户健康数据的持续跟踪和评估。
